# Go Tutorial by Edureka

## Introduction

This repository contains code I've written as I followed along the [Edureka's Golang Tutorial For Beginners on their YouTube channel](https://www.youtube.com/watch?v=Q0sKAMal4WQ)

---

## Architecture

This code in this repository is structured in the following way:

`<directory>/main.go`

---

## Usage of the project

### Requirements

* [Go installed on your system](https://go.dev/dl/)

### Execution

To run the programs of this repository, run the following command:

`go run <directory>/main.go`

For example,

`go run 018-webserver/main.go`
